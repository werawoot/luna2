# Luna Smart Home

Luna Smart Home is a demo PHP application for managing IoT devices, usage data, and billing in a household. It provides a simple API and web interface to interact with smart devices and monitor real-time statistics.

## Installation

1. Clone the repository.
2. Copy `config/config.sample.php` to `config/config.php` and adjust the settings.
3. Install dependencies using Composer.
4. Import the SQL files from the `sql/` directory into your MySQL database.
5. Configure your web server to serve files from the `public/` directory.
6. Start the built-in PHP server with `php -S localhost:8000 -t public`.
